# 💱 Forex Time Machine

> See how currency rates changed over time — enter any date range and compare!

## Features
- Historical exchange rates for any date range
- Compare any 2 currencies
- Interactive line chart
- Download data as CSV

## Run Locally
pip install streamlit requests pandas
streamlit run app.py

## Data Source
frankfurter.app — free, no API key needed

## Built With
Python · Streamlit · Pandas
